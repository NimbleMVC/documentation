Middleware

Klasa ładujące skrypty użytkownika do kernel'a

Namespace

Klasa:

Nimblephp\framework\Middleware

Interfejs:

Nimblephp\framework\Interfaces\MiddlewareInterface

Metody

afterBootstrap

public function afterBootstrap()

Kod wykonywany po załadowaniu kodu przez bootstrap

beforeController

public function beforeController(string $controllerName, string $action, array $params)

Kod wykonywany przed załadowaniem kontrolera

afterController

public function afterController(string $controllerName, string $action, array $params)

Kod wykonywany po załadowaniu kontrolera

handleException

public function handleException(Throwable $exception)

Kod wykonuje się na końcu handleException, po zapisaniu logu

Przykłady

Aby użyć klasy należy w głównym folderze utworzyć plik Middleware.php o następującym szablonie klasy:

<?php

class Middleware extends \Nimblephp\framework\Middleware
{

}

lub z interfejsu

class Middleware implements \Nimblephp\framework\Interfaces\MiddlewareInterface
{

    public function afterBootstrap()
    {
    }

    public function beforeController(string $controllerName, string $action, array $params)
    {
    }

    public function afterController(string $controllerName, string $action, array $params)
    {
    }

    public function handleException(Throwable $exception)
    {
    }

}
