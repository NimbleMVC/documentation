    Form

    Generowanie formularzy

    Namespace

    Nimblephp\form

    Zmienne

    • protected array $fields = [] : Tablica pól formularza.
    • protected MethodEnum $method = MethodEnum::POST: Metoda HTTP formularza, domyślnie POST.
    • protected ?string $action = null: Adres URL docelowy formularza.
    • protected ?string $id = null: Identyfikator formularza.
    • protected Request $request: Instancja żądania HTTP.
    • protected array $data = []: Dane wejściowe formularza.
    • protected bool $addLinebreak = true: Czy dodawać znacznik przełamania linii po każdym polu.
    • protected array $validationErrors = []: Tablica błędów walidacji.
    • protected bool $addFormNode = true: Dodanie noda form
