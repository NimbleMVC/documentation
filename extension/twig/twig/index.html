Instancja Pomocnika Twig

Klasa odpowiada za zarządzanie środowiskiem renderowania szablonów Twig.

Namespace

Klasa:

Nimblephp\twig\Twig

Zmienne statyczne

  • $globalVariables = []: Zmienne globalne
  • $globalPaths = []: Zmienne z dodatkowymi ścieżkami do widoków

Zmienne

  • $twigFileSystemLoader: Instancja FilesystemLoader do zarządzania ścieżkami plików.
  • $twigEnvironment: Instancja Environment reprezentująca środowisko Twig.

Metody

addPath

public function addPath($path): void
  • Dodaje ścieżkę do loadera systemu plików.

render

public function render(string $twigFilePath, array $variables = []): string

Renderuje widok Twig z określonymi zmiennymi i zwraca wynikowy ciąg.
