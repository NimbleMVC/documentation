Session

Session to klasa odpowiedzialna za zarządzanie sesjami użytkowników. Umożliwia inicjalizację sesji, a także ustawianie, pobieranie, sprawdzanie istnienia, usuwanie oraz niszczenie danych sesji.

Namespace

Klasa:

Nimblephp\framework\Session

Interfejs:

Nimblephp\framework\Interfaces\SessionInterface

Metody

Konstruktor

public function __construct()

Inicjalizuje sesję, jeśli jeszcze nie została uruchomiona.

set

public function set($key, $value): void

Ustawia wartość w sesji pod określonym kluczem.

get

public function get($key): mixed

Pobiera wartość z sesji dla podanego klucza, zwraca null jeśli klucz nie istnieje.

exists

public function exists($key): bool

Sprawdza, czy w sesji istnieje wartość przypisana do danego klucza.

remove

public function remove($key): void

Usuwa element z sesji pod określonym kluczem.

destroy

public function destroy(): void

Niszczy wszystkie dane sesji i kończy sesję.

regenerate

public function regenerate(?bool $removeOldSession = false): void

Regeneruje ID sesji, opcjonalnie usuwając stare dane sesji.
