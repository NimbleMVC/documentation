Cookie

Cookie to komponent odpowiedzialny za zarządzanie ciasteczkami w aplikacji. Zapewnia metody do ustawiania, pobierania, sprawdzania istnienia i usuwania ciasteczek, z możliwością konfiguracji ich bezpieczeństwa i innych opcji.

Namespace

Klasa:

Nimblephp\framework\Cookie

Interfejs:

Nimblephp\framework\Interfaces\CookieInterface

Zmienne

  • protected static bool $defaultSecure: Domyślne ustawienie bezpiecznego przesyłania ciasteczek (true albo false).
  • protected static bool $defaultHttpOnly: Domyślne ustawienie dostępności ciasteczka tylko przez HTTP (true albo false).
  • protected static string $sameSite: Domyślna polityka dla atrybutu SameSite ciasteczka ('Lax', 'Strict', lub 'None').

Metody

setSameSite

public static function setSameSite(string $sameSite): void

Ustawia politykę SameSite dla ciasteczek.

setDefaultSecure

public static function setDefaultSecure(bool $defaultSecure): void

Ustawia domyślne bezpieczeństwo przesyłania ciasteczek.

setDefaultHttpOnly

public static function setDefaultHttpOnly(bool $defaultHttpOnly): void

Ustawia domyślne ustawienie dostępności ciasteczka tylko przez HTTP.

set

public function set(string $name, mixed $value, int $expire = 0, string $path = '/', string $domain = '', ?bool $secure = null, ?bool $httponly = false): void

Ustawia ciasteczko z określoną nazwą, wartością, czasem wygaśnięcia, ścieżką, domeną, bezpieczeństwem i dostępnością HTTP.

get

public function get(string $name): mixed

Pobiera wartość ciasteczka o danej nazwie, zwraca null jeśli ciasteczko nie istnieje.

exists

public function exists(string $name): bool

Sprawdza, czy ciasteczko o podanej nazwie istnieje.

remove

public function remove(string $name): void

Usuwa ciasteczko o podanej nazwie.
