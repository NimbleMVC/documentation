Model

Model to główna część systemu odpowiadająca za logikę biznesową i zarządzanie danymi. Jego zadaniem jest przechowywanie danych, ich przetwarzanie i dostarczanie ich w odpowiedniej formie do kontrolera.

Namespace

Klasa:

Nimblephp\framework\Model

Abstract:

Nimblephp\framework\Abstracts\AbstractModel

Interface:

Nimblephp\framework\Interfaces\ModelInterface

Zmienne

  • public null|string|false $useTable: Nazwa tabeli w bazie danych używana przez model, null oznacza automatyczne ustalenie nazwy, false deaktywuje działanie z bazą danych.
  • public string $name: Nazwa modelu.
  • protected Table $table: Instancja tabeli obsługującej operacje na bazie danych.
  • protected ?int $id: Aktualne ID elementu w bazie danych.

Metody

create

public function create(array $data): bool

Tworzy nowy rekord w bazie danych na podstawie dostarczonych danych. Rzuca DatabaseException jeśli operacja nie powiedzie się.

read

public function read(?array $condition = null, ?array $columns = null, ?string $orderBy = null): array

Zwraca pojedynczy rekord na podstawie podanych warunków. Rzuca DatabaseException przy błędzie.

readAll

public function readAll(?array $condition = null, ?array $columns = null, ?string $orderBy = null, ?string $limit = null, ?string $groupBy = null): array

Zwraca listę rekordów spełniających podane warunki. Rzuca DatabaseException przy błędzie.

update

public function update(array $data): bool

Aktualizuje rekord o danym ID, używając dostarczonych danych. Rzuca DatabaseException jeśli operacja nie powiedzie się.

delete

public function delete(): bool

Usuwa rekord o danym ID. Rzuca DatabaseException jeśli operacja nie powiedzie się.

getId

public function getId(): ?int

Zwraca ID aktualnie zarządzanego elementu.

setId

public function setId(?int $id = null): ModelInterface

Ustawia ID dla aktualnie zarządzanego elementu, zwraca samą siebie.

prepareTableInstance

public function prepareTableInstance(): void

Przygotowuje instancję tabeli, na której będą wykonywane operacje.
