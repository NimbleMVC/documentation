Config

Klasa służy do zarządzania konfiguracją aplikacji, umożliwiając ładowanie, zapisywanie i pobieranie wartości konfiguracyjnych.

Namespace

Klasa:

Nimblephp\framework\Config

Zmienne

  • protected static array $config: Przechowuje wszystkie wartości konfiguracyjne jako tablica asocjacyjna.

Metody

get

public static function get(string $name, mixed $default = null): mixed

Pobiera wartość konfiguracji o podanej nazwie. Jeśli wartość nie istnieje, zwraca wartość domyślną.

set

public static function set(string $name, string $value): void

Ustawia wartość konfiguracji o podanej nazwie.

loadFromEnv

public static function loadFromEnv(string $filePath): bool

Ładuje konfiguracje z pliku środowiskowego. Zwraca true jeśli załadowanie się powiodło, w przeciwnym razie false.

processEnvContent

protected static function processEnvContent(string $content): void

Przetwarza pojedynczą linię pliku środowiskowego, parsując nazwę i wartość, a następnie zapisuje je do tablicy konfiguracji.

parseEnvValue

protected static function parseEnvValue(mixed $value): mixed

Parsuje wartość podaną jako string do odpowiedniego typu danych, np. liczby, wartości logiczne, czy ciągi znaków.
