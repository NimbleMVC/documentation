Funkcja akcji

Funkcja pozwala na wykonanie zapytania do kontrolera bezpośrednio w twig, pozwala to na doładowywania osobnych widoków do już istniejącego wydoku.

Metoda

action(string $controller, string $method, string ...$params): false|string

Zmienne

  • $controller: nazwa kontrolera
  • $method: nazwa metody
  • ...$params: dodatkowe przesłane parametry

Przykłady

index.twig

Table:
{\{ action('tables', 'main') }}
