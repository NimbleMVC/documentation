    Instancja Widoku Twig

    Klasa View jest zaprojektowana do zarządzania renderowaniem widoków za pomocą silnika szablonów Twig.

    Namespace

    Klasa:

    Nimblephp\twig\View

    Interface:

    Nimblephp\framework\Interfaces\ViewInterface

    Właściwości

    • Ścieżka widoku: protected string $viewPath = '/src/View/'
    • Zmienne widoku: protected array $variables = []
    • Instancja Twig: protected Twig $twig
    • Kod odpowiedzi: protected int $responseCode = 200

    Konstruktor

    • Przyjmuje instancję Twig i inicjalizuje klasę.

    Metody

    setResponseCode

    public function setResponseCode(int $responseCode): void

    Definiowanie kodu odpowiedzi.

    render

    public function render(string $viewName, array $data = []): void

    Renderowanie widoku na podstawie nazwy widoku i dostarczonych danych.
