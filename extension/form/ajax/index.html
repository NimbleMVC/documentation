Załądowanie pliku js

Należy zaimportować do widoku automatycznie tworzony plik ajax.js znajdujący się pod świeżką: /assets/form.js

Triggery

Kliknięcie w submit

ajaxform.submit

Przykład

$('#myForm').on('ajaxform.success', function(event, response, form, preventDefault) {
    ...
    preventDefault(); // Zatrzymuje domyślne zachowanie, np. przekierowanie
});

Sukcess

Wywoływany również gdy wystąpi walidacja ajaxform.success

Błąd

ajaxform.error
