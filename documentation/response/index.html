Response

Klasa służy do obsługi odpowiedzi HTTP.

Namespace

Klasa:

Nimblephp\framework\Response

Interface:

Nimblephp\framework\Interfaces\ResponseInterface

Zmienne

  • protected mixed $content: Treść odpowiedzi, może być różnego typu.
  • protected int $statusCode = 200: Kod statusu HTTP odpowiedzi, domyślnie 200.
  • protected array $headers = []: Tablica nagłówków HTTP odpowiedzi.
  • protected string $statusText = '': Tekstowy opis kodu statusu HTTP.
