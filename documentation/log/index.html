Log

Klasa służy do logowania zdarzeń w aplikacji.

Namespace

Klasa:

Nimblephp\framework\Log

Zmienne

  • public static string $session: Unikatowy identyfikator sesji, generowany dla każdej sesji logowania.

Metody

log

public static function log(string $message, string $level = 'INFO', array $content = []): bool

Metoda do zapisywania logów. Pozwala na zapisanie wiadomości logu, poziomu logowania oraz dodatkowych treści. Log jest zapisywany tylko jeśli w konfiguracji aplikacji włączone jest logowanie. Generuje unikatowy identyfikator sesji, jeśli nie istnieje. Formatuje i zapisuje dane logu w pliku JSON w katalogu logów projektu. Zwraca true, jeśli zapis się powiódł, lub false w przypadku niepowodzenia.
