Request

Klasa służy do obsługi żądań HTTP.

Namespace

Klasa:

Nimblephp\framework\Request

Interfejs:

Nimblephp\framework\Interfaces\RequestInterface

Zmienne

  • private array $query: Dane przesłane metodą GET.
  • private array $post: Dane przesłane metodą POST.
  • private array $cookies: Ciasteczka HTTP żądania.
  • private array $files: Pliki przesłane w ramach żądania.
  • private array $headers: Nagłówki HTTP żądania.
  • private array $server: Informacje o serwerze i środowisku wykonania żądania.
