Response

Klasa służy do obsługi odpowiedzi HTTP.

Namespace

Klasa:

Nimblephp\framework\Response

Interface:

Nimblephp\framework\Interfaces\ResponseInterface

Zmienne

  • protected mixed $content: Treść odpowiedzi, może być różnego typu.
  • protected int $statusCode = 200: Kod statusu HTTP odpowiedzi, domyślnie 200.
  • protected array $headers = []: Tablica nagłówków HTTP odpowiedzi.
  • protected string $statusText = '': Tekstowy opis kodu statusu HTTP.
  • protected RequestInterface $request: Instancja klasy Request

Metody

__construct

__construct()

Konstruktor klasy, nie jest zdefiniowany w podanym kodzie.

getContent

public function getContent(): string

Pobranie treści odpowiedzi.

setContent

public function setContent(mixed $content): void

Ustawienie treści odpowiedzi.

getStatusCode

public function getStatusCode(): int

Pobranie kodu statusu HTTP.

setStatusCode

public function setStatusCode(int $code, string $text = ''): void

Ustawienie kodu statusu HTTP i opcjonalnie tekstowego opisu.

addHeader

public function addHeader(string $name, string $value): void

Dodanie nagłówka HTTP do odpowiedzi.

send

public function send(bool $flush = false): void

Wysłanie odpowiedzi do klienta, wraz z nagłówkami i treścią.

$flush - pozwala od razu zwrócić treść wraz z exitem i wyczyszczeniem poprzedniej treści

redirect

public function redirect(string $url, int $statusCode = 302): never

Przekierowanie na inną stronę.

Jeżeli zapytanie jest ajaxowe, zwracana treść ze statusem 200 to:

{
  "type": "redirect",
  "url": "<url>"
}
