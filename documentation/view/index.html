View

Wskazówka: Domyślny widok opiera się na plikach phtml. Jeśli potrzebujesz bardziej zaawansowanych widoków, wkrótce pojawi się instrukcja i kod dla twig i innych.

Widok odpowiada za prezentację danych użytkownikowi. Jest to komponent, który wyświetla dane, które otrzymuje z kontrolera, zazwyczaj w postaci graficznego interfejsu użytkownika.

Namespace

Klasa:

Nimblephp\framework\View

Interface:

Nimblephp\framework\Interfaces\ViewInterface

Zmienne

  • protected string $viewPath = '/src/View/': Ścieżka do folderu z widokami.
  • protected int $responseCode = 200: Kod odpowiedzi.

Metody

setResponseCode

public function setResponseCode(int $responseCode): void

Definiowanie kodu odpowiedzi.

render

public function render(string $viewName, array $data = []): void

Renderowanie widoku na podstawie nazwy widoku i dostarczonych danych.
